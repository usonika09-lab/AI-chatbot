## Project Description
This project is a simple AI chatbot built using FastAPI and Streamlit.
The chatbot answers basic questions related to Bangalore, such as its location,
IT hubs, and what the city is known for.

## Technologies Used
- Python
- FastAPI (Backend)
- Streamlit (Frontend)

## Project Structure
AI_Chatbot/
- backend/
- frontend/
- screenshots/
- README.md

## How to Run the Project

### Step 1: Run Backend
uvicorn backend.main:app --reload

### Step 2: Run Frontend
streamlit run frontend/app.py

## Features
- Backend API using FastAPI
- Interactive frontend using Streamlit
- Topic-based chatbot with predefined responses
